CIFAR-10 Image Classifier
-----------------------------------------------------------
О проекте
-----------------------------------------------------------
Интерактивный сервис для классификации изображений из набора данных CIFAR-10.

Проект включает модель на PyTorch, интерактивный веб-интерфейс на Streamlit и деплой в публичный доступ.

-----------------------------------------------------------
Структура
-----------------------------------------------------------
streamlit_app.py – Интерфейс приложения.
train.py – Обучение модели.
requirements.txt – Используемые библиотеки в проекте.
models/cifar_cnn.py – Архитектура CNN.
utils/data.py – Загрузка данных и трансформации.
artifacts/cifar_net.pth – Обученная модель.
